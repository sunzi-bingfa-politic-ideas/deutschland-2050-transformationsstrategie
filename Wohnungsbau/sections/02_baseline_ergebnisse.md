## 2.5 Investitionsbedarf und ROI

### Investitionsstruktur (GCADI-Szenario)

| Investitionsposten | Kosten | Zeitraum |
|--------------------|--------|----------|
| Modulbau-Fabriken (~30 Stueck) | 6,0 Mrd. EUR | Jahre 1-5 |
| 3D-Druck-Equipment | 4,0 Mrd. EUR | Jahre 1-5 |
| Robotik-Integration | 2,5 Mrd. EUR | Jahre 1-5 |
| Ausbildung/Umschulung | 1,0 Mrd. EUR | Jahre 1-5 |
| F&E und Pilotprojekte | 1,0 Mrd. EUR | Jahre 1-3 |
| **Gesamt** | **14,5 Mrd. EUR** | 5 Jahre |

### Return on Investment

| Kennzahl | Wert |
|----------|------|
| Gesamtinvestition | 14,5 Mrd. EUR |
| Jaehrliche Ersparnis (ab Jahr 5) | 91,3 Mrd. EUR |
| ROI | **0,4 Jahre (5 Monate)** |
| Kumulative Ersparnis (20 Jahre) | 793 Mrd. EUR |
| Multiplikator | **55x** |

**Zentrale Erkenntnis**: Jeder investierte Euro bringt ueber 20 Jahre einen Return von 55 Euro. Die Investition amortisiert sich in weniger als 6 Monaten.

## 2.7 Zusammenfassung der Baseline-Ergebnisse

| Erkenntnis | Detail |
|------------|--------|
| Defizit in 5 Jahren schliessbar | Mit GCADI statt 7 Jahren (Status Quo) |
| Kostenreduktion **-54%** | Von 2.500 auf 1.156 EUR/m² |
| Investitionsbedarf | 14,5 Mrd. EUR (einmalig, 5 Jahre) |
| Kumulative Ersparnis | 793 Mrd. EUR (20 Jahre) |
| ROI | 0,4 Jahre (extrem schnell) |
| Arbeitskraefte-Entlastung | 157.248 Stellen/Jahr |
| Sozialwohnungs-Ersparnis | 85 Mrd. EUR |

**Zentrale Erkenntnis**: Die GCADI-Reform ist oekonomisch hochattraktiv. Mit einer Investition von 14,5 Mrd. EUR kann Deutschland:
1. Das Wohnungsdefizit in 5 statt 7 Jahren schliessen
2. Die Baukosten halbieren
3. 793 Mrd. EUR ueber 20 Jahre einsparen
4. Den Fachkraeftemangel durch Automatisierung entschaerfen
5. Das Sozialwohnungsdefizit fuer 85 statt 171 Mrd. EUR schliessen
